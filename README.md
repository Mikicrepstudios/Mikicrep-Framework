# Mikicrep Framework
## By Mikicrep

This is framework written in C++ and uses SDL2 as GUI library.
This framework allows you easier writing of SDL code.

## Setuping env
- Extract all sdl2mingw .zip files
- Extract all output/dll .zip files in output directory
